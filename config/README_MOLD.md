# 模具工业术语字典说明

## 概述

`moju_dict.json` 包含了模具工业领域的专业术语，主要涵盖线切割加工相关术语。

## 术语列表

### 慢走丝线切割

- **慢丝割一刀**：慢走丝线切割加工的第一刀切割
- **慢丝割一修一**：慢走丝线切割加工的第一次修刀
- **慢丝割一修二**：慢走丝线切割加工的第二次修刀
- **慢丝割一修三**：慢走丝线切割加工的第三次修刀

### 快走丝线切割

- **快丝割一刀**：快走丝线切割加工的第一刀切割

### 中走丝线切割

- **中丝割一修一**：中走丝线切割加工的第一次修刀

## 使用方法

这些术语已集成到 CodeWhisper 的词典系统中，在语音转录时会自动识别和修正。

### 常见误识别

由于语音识别的特点，这些术语可能被误识别为：

- "慢思割" → "慢丝割"
- "快思割" → "快丝割"
- "一道" → "一刀"
- 数字形式："一修1" → "一修一"

系统会自动将这些误识别修正为正确的术语。

## 扩展

如需添加更多模具术语，请编辑 `config/moju_dict.json` 文件，按照现有格式添加新术语及其可能的误识别变体。
