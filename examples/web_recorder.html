使用指南

  1. 配置服务器 - 填写 CodeWhisper API 服务器的地址和端口（已预设）
  2. 测试连接 - 点击"测试连接"确保服务器可访问
  3. 选择设置 - 选择模型大小和语言（默认已配置）
  4. 开始录音 - 点击"🎙️ 开始录音"按钮或按空格键
  5. 实时转录 - 录音过程中会自动显示转录结果（每 3 秒更新）
  6. 停止录音 - 说完后点击"⏹️ 停止录音"或按空格键
  7. 最终结果 - 停止后会进行最终转录，结果自动复制到剪贴板
